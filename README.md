# Programação Orientada a Objetos em Java

[English version](./README-en.md)

## 📌 Sobre o Repositório

Este repositório contém os códigos desenvolvidos por mim durante a disciplina de **Programação Orientada a Objetos (POO)**, no curso técnico em Desenvolvimento de Sistemas pelo **COLTEC-UFMG**.

Os programas foram desenvolvidos em **Java**, uma linguagem de programação robusta, fortemente tipada e orientada a objetos, amplamente utilizada no mercado para desenvolvimento de sistemas corporativos, aplicações web e mobile.

Neste repositório, **cada pasta (E01, E02, etc.) corresponde a um módulo de estudo**, abordando desde os fundamentos da linguagem até conceitos avançados como Polimorfismo, Collections e Serialização.

---

## ▶️ Como executar os projetos

### 1) Pré-requisitos

- Certifique-se de ter o **JDK (Java Development Kit)** instalado (versão 17 ou superior recomendada).
- É recomendado utilizar uma IDE como **IntelliJ IDEA** (já que o projeto contém arquivos de configuração `.iml`) ou **VS Code** com o Extension Pack for Java.

### 2) Abrindo um projeto

- Abra a sua IDE de preferência.
- Vá em **Arquivo > Abrir...** e selecione a pasta raiz do repositório ou a pasta específica do exercício (ex: `E10-Serializacao-main`).

### 3) Executando o código

- Localize o arquivo principal dentro da pasta `src` (geralmente nomeado como `Main.java` ou contendo o método `public static void main`).
- Clique no botão **Run (▶️)** da sua IDE para compilar e executar o programa.
